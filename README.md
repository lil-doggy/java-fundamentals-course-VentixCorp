# Fundamentos de Java para Principiantes

## 🧠 Resumen del Curso

Este curso introduce a estudiantes de secundaria a los fundamentos de la programación con Java, incluyendo programación orientada a objetos (POO), a través del desarrollo de un juego interactivo de adivinanzas.

El aprendizaje es práctico, progresivo y completamente en línea, sin necesidad de instalaciones.

---

**Duración total**: ~60 minutos  
**Público objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación  
**Prerrequisitos**: Ninguno  
**Herramientas necesarias**: Solo navegador web (Chrome, Firefox, Safari, Edge)

---

📂 **Repositorio de Código Fuente**:  
https://github.com/lil-doggy/java-fundamentals-course-VentixCorp

### Lección 6: Proyecto final – Juego de adivinanzas (10 minutos)

- **Descripción**: Construcción del juego completo usando POO.
- **Enlace**: https://youtu.be/PS4cv5ppqgE
- **Conclusiones clave**:
  - Encapsulación
  - ArrayList
  - Flujo del juego por niveles:
  El juego está organizado de la siguiente manera:
```
Bienvenida
   ↓
Ingreso de nombre
   ↓
Nivel Fácil
   ↓
Nivel Intermedio
   ↓
Nivel Difícil
   ↓
Pantalla final
```
Además, cada nivel contine 3 adivinanzas que el jugador debe resolver:
```
Mostrar pregunta
   ↓
Jugador responde
   ↓
¿Correcto?
   ├── Sí → +10 puntos
   └── No → Mostrar pista → segundo intento
                 ↓
            ¿Correcto?
               ├── Sí → +10 puntos
               └── No → -1 vida
```
- **Proyecto final online**:  

```
OnlineGDB: https://onlinegdb.com/D5lk1ntLB
GitHub: https://github.com/lil-doggy/java-fundamentals-course-VentixCorp.git
```
